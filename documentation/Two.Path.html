Class: Path

Two.Path(verticesopt, closedopt, curvedopt, manualopt)

new Path(verticesopt, closedopt, curvedopt, manualopt)

This is the base class for creating all drawable shapes in two.js. Unless specified methods return their instance of Two.Path for the purpose of chaining.
Parameters:
Name Type Attributes Default Description
vertices Array.<Two#Anchor> <optional>
 A list of Two.Anchors that represent the order and coordinates to construct the rendered shape.
closed Boolean <optional>
 false Describes whether the shape is closed or open.
curved Boolean <optional>
 false Describes whether the shape automatically calculates bezier handles for each vertex.
manual Boolean <optional>
 false Describes whether the developer controls how vertices are plotted or if Two.js automatically plots coordinates based on closed and curved booleans.
