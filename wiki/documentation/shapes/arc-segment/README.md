### Constructor


| Argument | Description |
| ---- | ----------- |
| `x` | The x position of the arc segment. |
| `y` | The y position of the arc segment. |
| `innerRadius` | The inner radius value of the arc segment. |
| `outerRadius` | The outer radius value of the arc segment. |
